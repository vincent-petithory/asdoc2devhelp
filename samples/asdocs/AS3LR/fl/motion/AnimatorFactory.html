Classpublic class AnimatorFactory
InheritanceAnimatorFactory AnimatorFactoryBase Object

Language Version: ActionScript 3.0
Runtime Versions: AIR 1.0 Flash Player 9.0.28.0

The AnimatorFactory class provides ActionScript-based support to associate one Motion object with multiple display objects.

Use the AnimatorFactory constructor to create an AnimatorFactory instance. Then, use the methods inherited from the AnimatorFactoryBase class to associate the desired properties with display objects.
