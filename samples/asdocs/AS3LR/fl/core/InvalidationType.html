Classpublic class InvalidationType
InheritanceInvalidationType Object

Language Version: ActionScript 3.0
Runtime Versions: AIR 1.0 Flash Player 9.0.28.0

The InvalidationType class defines InvalidationType constants that are used by the type property of an event object that is dispatched after a component is invalidated. These constants are used by component developers to specify the portion of the component that is to be redrawn after the component is invalidated.
