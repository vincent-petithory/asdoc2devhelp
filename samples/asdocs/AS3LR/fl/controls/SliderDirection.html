HORIZONTALproperty
public static var HORIZONTAL:String = "horizontal"

Language Version: ActionScript 3.0
Runtime Versions: AIR 1.0 Flash Player 9.0.28.0

Position the Slider component on the horizontal axis.


Example
The following example creates a horizontal slider: 
import fl.controls.Slider;
import fl.controls.SliderDirection;
var hSlider:Slider = new Slider();
hSlider.direction = SliderDirection.HORIZONTAL;
hSlider.snapInterval = 1;
hSlider.tickInterval = hSlider.snapInterval;
hSlider.move(10, 10);
addChild(hSlider);
