setlocale()method 
public function setlocale(category:uint, locale:String = null):String

The setlocale() method encapsulates all the logic to set and retrieve the current locale.

The default locale for a given category is determined in the following order :

  • the value assigned to the category, if any.
  • the value of the global LANGUAGE variable, if set.
  • the value of the LANG const, which defaults to the language Flash Player determines (see the LANG constant for more informations).

Parameters

category:uint — the category to set/retrieve information.
 
locale:String (default = null) — the locale to set. If not null, the specified locale will be set for the specified category. If null, the method will return the current locale for the specified category. Use the mklocale() function to create a standard locale.

Returns
String

LANGConstant
public const LANG:String

The native locale of the system, which defaults to the language Flash Player determines (see the table below).

As Flash Player only determines a language code, not a full locale, a locale is made out of that language code. When Flash Player encounters an unknown locale, then en_US is used by default.
